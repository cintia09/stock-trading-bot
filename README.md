# 模拟股票交易系统

## 账户信息
- **初始资金**：1,000,000 元
- **开始日期**：2026-02-06
- **交易规则**：遵循中国A股 T+1 规则

## 交易规则
1. **T+1 制度**：当天买入的股票，第二天才能卖出
2. **交易时间**：周一至周五 9:30-11:30, 13:00-15:00
3. **涨跌停限制**：主板 ±10%，科创板/创业板 ±20%
4. **最小交易单位**：100股（1手）
5. **交易费用**：
   - 佣金：万分之2.5（最低5元）
   - 印花税：卖出时千分之1
   - 过户费：万分之0.2

## 自动化任务
- **开盘操作**：每个交易日 9:35（开盘后5分钟）自动执行交易
- **收盘总结**：每个交易日 15:05（收盘后5分钟）自动总结

## 文件结构
- `account.json` - 账户状态（资金、持仓）
- `transactions.json` - 交易记录
- `watchlist.json` - 关注股票列表
- `strategy.md` - 交易策略说明
- `daily-log/YYYY-MM-DD.md` - 每日交易日志

## 操作流程
1. 每个交易日开盘前：获取市场数据，分析机会
2. 开盘后（9:35）：自动执行买卖操作
3. 收盘后（15:05）：自动记录当日盈亏，更新持仓市值
