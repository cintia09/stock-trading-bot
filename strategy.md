# 模拟股票交易策略 v2.1

## 自动复盘机制 (New!)

### 每日复盘流程 (15:30自动执行)
1. **盈亏统计**：今日盈亏、胜率、盈亏比
2. **交易分析**：每笔交易的问题识别
3. **策略诊断**：止损执行、选股逻辑、仓位管理
4. **自动调参**：根据复盘结果自动调整策略参数

### 策略参数自动调整规则
- 止损执行不及时 → 收紧止损线
- 盈利过早卖出 → 放宽止盈阈值
- 胜率低于40% → 提高选股评分门槛
- 仓位过重 → 降低单只最大仓位

### 复盘文件
- 每日复盘: `/stock-trading/reviews/YYYY-MM-DD.json`
- 策略参数: `/stock-trading/strategy_params.json`
- 参数自动版本管理，可追溯历史变化

## 核心交易理念

### 成功交易员经验总结
1. **惯性交易法**：处于启动形态的股票，惯性上涨概率大，盈利2-3%就走
2. **趋势跟踪**：顺势而为，不逆势操作
3. **资金管理**：永远不满仓，保留30%现金应对机会
4. **止损纪律**：单笔亏损不超过8%，坚决执行

### 短线操作技巧
1. **买入时机**：
   - 股价站上5日均线，且5日均线向上
   - 成交量较前日放大20%以上
   - MACD金叉或即将金叉
   - 板块轮动到位，资金流入

2. **卖出时机**：
   - 盈利3-5%时减仓50%
   - 跌破5日均线且无反弹迹象
   - 成交量萎缩，上涨乏力
   - 板块退潮，资金流出

### 量化信号
- **买入信号**：价格>MA5, MA5>MA10, 成交量>昨日1.2倍
- **卖出信号**：价格<MA5, 或盈利>5%, 或亏损>8%

## 仓位管理

### 初始建仓计划
- 总资金：100万
- 首次建仓：40-50万（4-5成仓位）
- 单只股票：不超过15万（15%）

### 动态调仓
- 盈利时：浮盈5%减仓30%，浮盈10%再减30%
- 亏损时：亏损5%观察，亏损8%清仓

## 选股逻辑

### 主线方向（2026年2月）
1. **贵金属** - 金价创新高，紫金矿业等龙头受益
2. **消费** - 春节旺季+政策刺激，茅台等确定性强

## 每日操作流程

### 盘前（9:00-9:25）
1. 获取关注股票的开盘价和集合竞价情况
2. 检查持仓股票的隔夜消息
3. 确定当日操作计划

### 盘中（9:30-15:00，每5分钟）
1. 获取实时价格
2. 检查买卖信号
3. 执行交易并记录
4. 实时通知用户

### 盘后（15:00-15:30）
1. 统计当日盈亏
2. 更新账户状态
3. 生成日报发送用户
4. 规划明日操作

## 风险控制

### 硬性规则
- 单日最大亏损：3万（3%）
- 单只最大仓位：15万（15%）
- 不追涨停板
- 不在尾盘15分钟买入
- 大盘跌幅>2%时不新开仓

### 市场判断
- 上证指数在MA5上方：正常操作
- 上证指数跌破MA5：谨慎操作，降低仓位
- 上证指数跌破MA20：观望为主
